# FormCraft — Interactive Form Builder UI

A beautifully polished, interactive drag-and-drop Form Builder UI built within a native Laravel Blade architecture[cite: 5, 72]. This application enables users to visually construct complex web forms by dragging input, choice, and location field types from a palette into a live canvas space, dynamically configure detailed validation parameters per field, and serialize the runtime data model into a standard JSON schema[cite: 2, 3, 12].

---

## 🚀 Features

- **Drag and Drop Canvas:** Smooth interaction mechanics for placing elements on the builder workspace[cite: 2].
- **Live Field Editing:** Select a placed field to dynamically change labels, placeholders, and rules instantly[cite: 33, 34].
- **Duplicate Elements:** Instantly copy an existing field with all its current configurations preserved.
- **Delete Fields:** Effortlessly clean up and remove unwanted field components from your canvas[cite: 38].
- **Reorder via Drag Handles:** Grab any element's move handle to shuffle and adjust layout sequencing[cite: 30].
- **JSON Schema Export:** Generate a structured, production-ready JSON data scheme with a single click[cite: 3, 21].
- **LocalStorage Persistence:** Form states survive accidental browser or page refreshes[cite: 49].
- **Responsive Design:** Optimized down to 1024px widths for flawless presentation on tablets and desktop monitors[cite: 46].
