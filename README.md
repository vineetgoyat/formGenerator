# FormCraft

## Setup

composer install

php artisan serve

## Features

- Drag and Drop
- Edit Fields
- Duplicate Fields
- Delete Fields
- Reorder Fields
- JSON Export
- Local Storage Persistence
- Responsive Design

## Sample JSON

{
  "title":"Student Registration Form",
  "fields":[
    {
      "type":"text",
      "label":"Name"
    }
  ]
}
